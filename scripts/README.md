# Assistant Juridique Marocain - Scripts de démarrage

## Scripts utiles

### Démarrage rapide
```bash
# Démarrer tous les services
./start.sh

# Arrêter tous les services
./stop.sh

# Voir les logs
./logs.sh
```

### Maintenance
```bash
# Recharger les données
./reload-data.sh

# Vider l'historique
./clear-history.sh

# Vérifier le statut
./status.sh
```

### Développement
```bash
# Installer les dépendances
./install.sh

# Tests
./test.sh

# Build production
./build.sh
```
